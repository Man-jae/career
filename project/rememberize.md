# 기억하기 (Rememberize)
> MVVM 패턴 및 모던 안드로이드 기술 학습을 위해 진행한 메모 앱 프로젝트 입니다.

## 기능

- 테마 설정 기능 (Light / Dark)
- Pin 기능
